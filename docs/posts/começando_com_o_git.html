Começando com o Git

Autor: Batman

Data: 12/11/2023

Deixando o Git conhecer você: Comando 1: git config --global user.name Seu nome Comando 2: git config --global user.email seu-email@example.com Iniciando o repositório Git local: git init Clonar repositório remoto do GitHub: git clone url_repositorio Verificar alterações dos arquivos: git status Adicionar arquivo para commit: git add nome_do_arquivo Adicionar todos arquivos da pasta para commit: git add . Fazer commit das mudanças: git commit -m 'Mensagem do commit' Enviar mudanças para o repositório remoto no GitHub: git push origin nome_da_branch Baixar alterações do repositório remoto: git pull origin nome_da_branch
